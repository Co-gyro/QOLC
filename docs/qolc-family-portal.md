# QOLC 利用者ポータル設計書

> 入居者本人・家族向けのモバイルファースト画面（/user）

---

## 1. 概要

### 対象ユーザー

入居者本人、入居者の家族
- LINEユーザー → LINE LIFF（Webビュー）
- LINE未使用者 → Webブラウザ

### アカウント種別

| 種別 | 説明 | アイコン |
|------|------|---------|
| 入居者本人（self） | 入居者ご本人のアカウント | 👤 |
| 家族（family） | 入居者のご家族のアカウント | 👥 |

- 各入居者につき決済担当者は1名のみ
- 決済担当者のみがカード登録・変更可能

### 主な機能

| 機能 | 説明 |
|------|------|
| 明細確認 | 月別の利用明細を確認 |
| 領収書ダウンロード | 領収書PDFをダウンロード |
| カード登録 | 決済用カードの登録・変更（決済担当者のみ） |
| 設定 | アカウント情報確認、通知設定 |

### 設計方針

| 方針 | 内容 |
|------|------|
| モバイルファースト | スマホでの利用を最優先 |
| シンプル | 必要最小限の機能に絞る |
| 高齢者対応 | 大きめの文字、分かりやすいUI |
| LINE/Web共通 | 同じ画面をLIFFとWebで共用 |

---

## 2. アクセス方法

### LINE経由（メイン：70-80%）

```
1. 施設からQRコード案内書を受け取る
2. QRコードを読み取り、LINE公式アカウント「QOLC」を友だち追加
3. リッチメニューの「明細を確認」をタップ
4. LINEログインで認証（初回のみ）
5. 利用者ポータルが表示される（LIFF）
```

### Web経由（サブ：15-25%）

```
1. 施設から招待メールを受け取る
2. メールのリンクをクリック
3. パスワードを設定（初回のみ）
4. Webブラウザでログイン
5. 利用者ポータルが表示される
```

## 5. UIデザイン方針

### カラー

| 用途 | カラー |
|------|--------|
| メインカラー | エメラルドグリーン（emerald-600 / #059669）施設ポータルと統一 |
| アクセント | ティール（#0d9488） |
| 成功 | 緑（#059669） |
| エラー | 赤（#dc2626） |
| 警告 | 黄（#d97706） |
| テキスト | ダークグレー（#1f2937） |
| 背景 | 薄いグレー（#f9fafb） |

### フォントサイズ（高齢者対応）

| 用途 | サイズ |
|------|--------|
| 見出し | 20-24px |
| 本文 | 16-18px |
| 補足 | 14px |
| 金額 | 24-32px（目立たせる） |

### タッチターゲット

- ボタン：最小 44x44px
- リスト項目：最小 48px高さ
- 十分な余白を確保

### レスポンシブ

| デバイス | 対応 |
|---------|------|
| スマホ（〜428px） | メイン対応（max-width: 428px、中央寄せ、影付き） |
| タブレット | 対応（スマホレイアウト拡大） |
| PC | 対応（中央寄せ、最大幅制限） |

## 6. 認証フロー

### LINE経由

```
リッチメニュー「明細を確認」タップ
    ↓
LIFF起動
    ↓
初回：LINEログイン認証
    ↓
Auth0がLINE IDとアカウント情報を照合
    ↓
アカウント種別（入居者本人/家族）を判定
    ↓
ホーム画面表示（種別に応じた挨拶）
    ↓
決済担当者でカード未登録の場合：カード登録モーダル表示
```

### Web経由

```
ログイン画面
    ↓
メールアドレス + パスワード入力
    ↓
Auth0認証
    ↓
アカウント種別（入居者本人/家族）を判定
    ↓
ホーム画面表示（種別に応じた挨拶）
    ↓
決済担当者でカード未登録の場合：カード登録モーダル表示
```

## 8. セキュリティ

### アクセス制御

- 自分（入居者本人/家族）が紐づいた入居者の情報のみ閲覧可能
- 他の入居者・家族の情報にはアクセス不可
- カード登録・変更は決済担当者のみ

### セッション

- 一定時間操作がない場合は自動ログアウト
- LINE経由の場合はLINEのセッションに依存

### カード情報

- QOLC側にカード番号を保持しない
- 表示は下4桁のみ（例：Visa •••• 1234）
- 登録・変更はUSEN PSP画面で実施
