# QOLC 介護施設ポータル設計書

> 介護施設向けの入居者管理・サービス提供者紐づけ画面

---

## 1. 概要

### 対象ユーザー

介護施設の管理者・スタッフ
- 施設管理者：全機能利用可能
- 施設スタッフ：閲覧・一部操作のみ

### 主な機能

| 機能 | 説明 |
|------|------|
| 入居者管理 | 入居者の登録・編集・削除、家族情報の管理 |
| サービス提供者管理 | 利用するサービス提供者の紐づけ |
| 明細確認 | 入居者の利用明細を確認 |
| 領収書設定 | 領収書の発行タイミング等を設定 |
| スタッフ管理 | 施設内スタッフアカウントの管理 |

---

### 3.2 入居者管理

#### 入居者一覧

**検索・フィルター**
- キーワード検索（氏名、被保険者番号）
- ステータス（入居中/退去済み）

**一覧テーブル**
| カラム | 説明 |
|--------|------|

| 被保険者番号 | 下4桁のみ表示 |
| 入居日 | 入居した日 |
| 家族登録 | 登録済み家族数 |

| カード登録 | 登録済み/未登録 |
| 操作 | 詳細を見る |

**操作ボタン**
- 「入居者を登録」ボタン

---

#### 入居者登録

**ステップ形式**

```
ステップ1: 入居者情報
    ↓
ステップ2: 家族情報
    ↓
ステップ3: 確認
    ↓
完了
```

**ステップ1: 入居者情報**
| 項目 | 必須 | 説明 |
|------|------|------|
| 氏名 | ○ | 姓・名 |
| フリガナ | ○ | セイ・メイ |
| 生年月日 | ○ | 年月日選択 |
| 性別 | ○ | 男性/女性 |
| 被保険者番号 | ○ | 介護保険被保険者番号 |
| 入居日 | ○ | 入居した日 |
| 部屋番号 | − | 施設内の部屋番号 |
| 備考 | − | 自由記述 |

**ステップ2: 家族情報**
| 項目 | 必須 | 説明 |
|------|------|------|
| 家族名 | ○ | 氏名 |
| 続柄 | ○ | 長男/長女/配偶者/その他 |
| 電話番号 | ○ | 連絡先 |
| メールアドレス | △ | LINE以外の場合必須 |
| 通知方法 | ○ | LINE/メール/郵送 |
| 請求先 | ○ | この家族が請求先か（はい/いいえ） |

- 「家族を追加」ボタンで複数家族を登録可能
- 少なくとも1名は必須

**ステップ3: 確認**
- 入力内容の確認
- 「登録」ボタン

**完了**
- 登録完了メッセージ
- LINEの場合：QRコード付き案内書を印刷するボタン
- 「入居者一覧へ」「続けて登録」ボタン

## 5. 権限

### 施設管理者

- すべての機能を利用可能
- スタッフの追加・編集・削除
- 設定の変更
- サービス提供者の紐づけ管理

### 施設スタッフ

- 入居者情報の閲覧・編集
- 家族情報の閲覧・編集
- 明細の閲覧
- サービス提供者の追加不可
- スタッフ管理不可
- 設定変更不可

---

## 6. 認証・セキュリティ

### 認証方式
- Auth0によるメール + パスワード認証
- MFA推奨

### アクセス制限
- 自施設のデータのみ閲覧・操作可能
- 他施設のデータにはアクセス不可

---

## 7. データフロー

### 入居者登録〜明細通知の流れ

```
1. 施設が入居者を登録
    ↓
2. 家族情報を登録（通知方法を設定）
    ↓
3. LINEの場合：QRコード案内書を家族に渡す
    ↓
4. 家族がLINE友だち追加・認証
    ↓
5. カード情報を登録（USEN PSP画面）
    ↓
6. サービス提供者が明細をアップロード
    ↓
7. 自動決済
    ↓
8. 家族に通知（LINE/メール）
    ↓
9. 月末に領収書発行
```

## 9. UIデザイン方針

### カラー
- メインカラー：グリーン系（#059669）運営者・提供者と区別
- 成功：緑（#059669）
- エラー：赤（#dc2626）
- 警告：黄（#d97706）

### レイアウト
- サイドバーは6項目
- 入居者登録はステップ形式で分かりやすく
- 入居者詳細はタブ形式で情報を整理

### レスポンシブ
- PC優先（業務利用想定）
- タブレット対応
- スマホは最低限

## 10. 入居者・家族アカウント発行

### LINE通知の場合

```
1. 入居者登録時に家族の通知方法を「LINE」に設定
    ↓
2. 登録完了後、QRコード付き案内書を生成
    ↓
3. 施設スタッフが案内書を印刷して家族に渡す
    ↓
4. 家族がQRコードを読み取りLINE友だち追加
    ↓
5. LINEログインで認証
    ↓
6. Auth0がLINE IDと入居者・家族を紐づけ
```

### メール通知の場合

```
1. 入居者登録時に家族の通知方法を「メール」に設定
    ↓
2. 登録完了後、招待メールが自動送信
    ↓
3. 家族がメールのリンクをクリック
    ↓
4. パスワードを設定
    ↓
5. Web画面にログイン可能に
```

### 郵送の場合

```
1. 入居者登録時に家族の通知方法を「郵送」に設定
    ↓
2. アカウント作成なし
    ↓
3. 明細・領収書は紙で郵送（施設が対応）
```
